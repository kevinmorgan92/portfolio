Java Programming

The module aims to introduce students to the concepts of object-oriented software development and demonstrate its practical application in problem-solving. It provides students with hands-on experience in writing computer programs, allowing them to gain practical skills in software development. Furthermore, the module introduces students to fundamental aspects of professional programming practice, equipping them with the necessary knowledge and skills to approach programming projects in a professional and systematic manner.

By covering these objectives, the module aims to empower students with a solid foundation in object-oriented programming, practical programming experience, and an understanding of professional programming practices.

The core topics of the module are as follows.

  • Java Basics - Primitive types, expressions, decision and looping contructs
  • Java objects, classes and methods - Anatomy of a class. Methods and contructors. Overloading
  • Parameter passing and exception handling - Parameter primitive types. Exception handling (try,catch,finally, throws etc)
  • File handling - Basic input/output operations
  • Collaborative classes - Building Java Programs with multiple classes. Message passing
  • Inheritance and polymorphism - Building inheritance hierarchies. Static and dynamic polymorphism
  • Stepwise refinement - Making the complicated simpler
  • Object-oriented design - Using UML. Static and dynamic modelling
  • Unit testing - Level of software testing. Junit with NetBeans

The module was made up of two assignments. The first assignment worth 40% was an Individual skill assessment done in class. A coding problem to be solved using skills learned throughout the module.
