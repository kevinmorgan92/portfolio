Interactive Web Authoring

The primary objective of this module was to introduce students to the fundamental concepts of HTML, CSS, and JavaScript. It aimed to equip them with the knowledge and skills necessary to build and maintain basic websites, including implementing essential functionalities like form validation and interactivity. Additionally, the module emphasized the importance of user experience (UX) design. It provided an understanding of how cognitive psychology influences user behavior and how to design user-friendly interfaces. Furthermore, the module addressed ethical considerations in web development and highlighted their significance. Students were exposed to ethical issues related to web design and development, encouraging them to prioritize ethical practices in their work. By covering these topics, the module aimed to provide students with a well-rounded understanding of web development principles, practical skills, UX design considerations, and ethical responsibilities.

This module provided learners with a comprehensive understanding of HTML, UX design, and ethical considerations in IT and AI. They gained proficiency in using IDs and classes within div elements to control layout and flow for multi-column views. Additionally, learners learned how to add interactivity to forms for validation and user input control. The module emphasized following conventional norms in website design, adopting a mobile-first approach. Furthermore, learners developed an understanding of the ethical implications and governance in IT and AI, enabling them to make responsible decisions. Overall, this module equipped learners with the knowledge and skills to create user-friendly websites, apply UX design principles, and navigate ethical challenges in the ever-evolving field of technology.

The core topics of the module are as follows.

  • HTML - The foundational concepts and syntax of HTML, to create the structure and layout of web pages.
  • CSS - The module covered CSS properties and selectors, how to style and format HTML elements to enhance the visual presentation of web pages.
  • Javascript - Knowledge of JavaScript programming concepts, including variables, functions, and events. How to add interactivity to web pages by manipulating the DOM and handling user interactions.
  • UX Design - The module emphasized the importance of considering the user's needs and preferences when designing websites. Introduction to UX design principles, such as user research, wireframing, prototyping, and usability testing, to create user-friendly and intuitive interfaces.
  • Mobile First Design - designing websites with a focus on mobile devices before adapting them for larger screens. Strategies for optimizing responsive design and ensuring a seamless user experience across different screen sizes.
  • Ethics - The module covered ethical considerations in IT and AI. Topics such as data privacy, security, fairness, and transparency.

The module was graded through two assignments. One being an individual assignment, a coding assignment which the student had to create a website with certain requirements to be filled. The other assignment was a group assignment to evaluate the UX design of a given website.
