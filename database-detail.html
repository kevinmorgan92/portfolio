Database Systems

This module had the objective of providing an explanation of the fundamental concepts involved in the design, structure, implementation, manipulation, and management of relational databases within modern information systems. It aimed to demonstrate the application of suitable models, processes, and techniques in designing, implementing, and utilizing database systems to meet specified business scenarios. Additionally, the module focused on developing skills necessary to construct databases based on given relational schemas using commercial database management systems. Students were also taught how to perform basic data manipulation and information retrieval operations using SQL. By covering these topics, the module aimed to equip students with the necessary knowledge and skills to effectively work with and utilize relational databases in real-world scenarios.

By completing this module, students will gain a comprehensive understanding of the theoretical foundations, structure, management, and operations of database systems. They will also gain insights into recent developments and future trends in the field. The module aims to equip students with the knowledge and skills to design databases using established techniques and implement these designs on a relational database management system. Students will also gain practical experience in data manipulation and query operations using SQL. Furthermore, they will have the opportunity to work with a commercial database management system and a web-based database management tool, providing them with valuable hands-on experience in real-world database environments. Overall, this module ensures that students develop a strong grasp of database concepts, practical skills in database design and implementation, and proficiency in utilizing database management systems and tools.

The core topics for this module were as follows.

  • Database Basics - Limitation of a file-based approach, Aims and benefits of a database approach, ANSI/SPARC three-level architecture, Database life cycle.
  • Data Models - Relational Model, Functional dependence, Normal Forms (First,Second,Third) Entity-relationship modelling, Mapping from ER model to relational schema
  • Query Processing in the Relational Model - Relational Algebra, SQL
  • Web Technology and DBMSs - Role of databases in web-based systems, Introduction to alternative DBMSs, including NoSQL databases and accessing DBMS via the web
  • Data Management Issues - Physical Storage of Data, Transaction management, Data Security, Legal ethical and social responsibilities.

The module was had five open book progress tests and a practical exam. I completed the module with a combined score of 84% and achieved an optional Certiport certification on IT Specialist Databases for my successful test results.
