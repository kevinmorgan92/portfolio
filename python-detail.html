Python Programming

This module aimed to introduce students to the core concepts of software development and its practical application in problem-solving. It provided hands-on experience in writing computer programs and familiarized students with essential aspects of professional programming practice. By focusing on both theoretical knowledge and practical skills, the module prepared students to apply software development principles effectively and adopt professional programming practices in their future endeavors.

In this module, the learner gained a comprehensive understanding of programming principles, including the importance of testing. They can now identify suitable programming constructs, solve simple problems, and apply logical thinking by breaking tasks into smaller steps. Additionally, they demonstrated proficiency in designing, developing, and testing code for readability, stability, and maintainability. Overall, the module provided a strong foundation in programming and problem-solving, equipping the learner with the skills to create efficient and well-designed code.

The module was structured as a blend of theory and practicals. With a focus on Procedural Programming via Python. The core topics covered are as follows:

  • Fundamentals - Overview of procedural programming
  • Data - Input, processing and output
  • Control Structures - Decision structures
  • Logic - Repetition structures
  • Structured Programming - Functions, passing and returning data
  • File IO - Handling files and exceptions
  • Debugging and Testing - Approaches and case studies
  • Data Structures - Lists, tuples, dictionaries and sets
  • Strings - File IO, Testing and Strings
  • External Libraries - Leveraging the use of existing code
  • Introduction to the OOP Paradigm - Transitioning from procedural programming
